# Agent Instructions

This document provides instructions for agents working on this codebase.

## GDPR and UK Data Sovereignty Compliance

The infrastructure for this project is designed to be compliant with GDPR and UK data sovereignty rules. All resources are deployed to the `europe-west2` (London) region, and data is not replicated outside of this region.

Please ensure that any new resources or changes to existing resources adhere to these compliance rules.
